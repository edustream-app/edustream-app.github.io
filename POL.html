Informations

PlayOnLinux utilise principalement le projet WineHQ. Si vous voulez les aider en obtenant un support professionnel pour wine, vous pouvez envisager d'acheter un produit codeweavers.

Nous aimerions aussi remercier les projets ScummVM et DOSBox

Version de développement

Pour obtenir la version de développement, ouvrez un terminal, et tapez :

git clone https://github.com/PlayOnLinux/POL-POM-4

Paquets

Toutes les distributions

Paquets génériques

Fichiers .tar.gz

Vos devez extraire ces fichiers et éxéctuer "./playonlinux".

PlayOnLinux : PlayOnLinux_4.3.4 .tar.gz

Archlinux

Arch Linux

Ajouter le dépôt Arch Linux FR

Ajoutez au fichier /etc/pacman.conf les lignes suivantes (architecture i386) :

[archlinuxfr]
Server = http://repo.archlinux.fr/i686

Architecture x86_64

[archlinuxfr]
Server = http://repo.archlinux.fr/x86_64

Installer PlayOnLinux

Tapez la commande suivante :

pacman -Syu playonlinux
Debian

Debian

Fichiers deb :

PlayOnLinux : PlayOnLinux_4.3.4 .deb

Utilisateurs de wheezy : Vous devez installer le paquet wine:i386 pour faire fonctionner PlayOnLinux

Avec le dépôt Stretch

Tapez les commandes suivantes :

wget -q "http://deb.playonlinux.com/public.gpg" -O- | apt-key add -
wget http://deb.playonlinux.com/playonlinux_stretch.list -O /etc/apt/sources.list.d/playonlinux.list
apt-get update
apt-get install playonlinux
